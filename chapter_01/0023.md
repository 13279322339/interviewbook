# 定义类 A 和类 B 如下：

```
class A {
    int a = 1;
    double d = 2.0;
    void show() {
        System.out.println("Class A: a=" + a + "\td=" + d);
    }
}

class B extends A {
    float a = 3.0f;
    String d = "Java program.";
    void show() {
        super.show();
        System.out.println("Class B: a=" + a + "\td=" + d);
    }
}
```

\(1\) 若在应用程序的main方法中有以下语句：

```
A a=new A();
a.show();
```

则输出的结果如何？

\(2\) 若在应用程序的main方法中定义类B的对象b：

```
A b=new B();
b.show();
```

则输出的结果如何？

# 参考解答

输出结果为;  
（1）Class A: a=1    d=2.0  
（2）Class A: a=1    d=2.0  
          Class B: a=3.0    d=Java program.  
解析：\(1\)
